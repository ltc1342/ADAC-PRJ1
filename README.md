# 1. Project Overview

## Hardware

### Main Controller

* STM32F411CEU6 (Black Pill)

### Communication Controller

* ESP32 WROOM DevKit

### Sensors

* DHT11 (Temperature & Humidity)
* BH1750 (Light Intensity)
* SMS-V1 LM393 (Soil Moisture)
* DS18B20 (Soil Temperature) *(Optional Enhancement)*

### Actuators

* Relay Pump
* Relay Mist Maker

### Display

* SH1106 OLED 128x64

### Communication

* UART STM32 ↔ ESP32
* WiFi
* MQTT
* Node-RED Dashboard

### Time Management

* STM32 RTC
